> 一些实用程序，用于简化 [vscode 扩展](https://marketplace.visualstudio.com/VSCode) 的开发

## 特性

- 支持使用 `package.nls.json` 用于国际化
- 其他常用方法

## 安装

```bash
# pnpm
pnpm add @tomjs/vscode-utils

# yarn
yarn add @tomjs/vscode-utils

# npm
npm add @tomjs/vscode-utils
```
